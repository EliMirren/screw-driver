ScrewDriver的目标是在IDE中生成类与数据库表或读取数据库表生成类的工作

核心接口(sd) 用于生成代码文件或数据库相关操作
接口包名org.mirrentools.sd
实现包名org.mirrentools.sd.impl
数据库操作工具接口(dbUtil) 用于正删改查数据库相关的操作
接口包名:org.mirrentools.sd.dbutil
实现类包名:org.mirrentools.sd.dbutil.impl
数据转换工具接口(converter) 主要用户实体类与数据库之间的转换
接口包名:org.mirrentools.sd.converter
实现类包名:org.mirrentools.sd.converter.impl.类型
数据描述类
类的描述包名:org.mirrentools.sd.models
数据库查询描述包名:org.mirrentools.sd.models.db.query
数据库创建描述包名:org.mirrentools.sd.models.db.update

生成代码执行流程

  1. 创建一个实体用于描述代码的类属性,对应:SdBean
  2. 添加要生成代码的模板,对应:Map(String, SdTemplate)
  3. 实例化生成代码的工具,对应:ScrewDriverCode
  4. 在ScrewDriverCode中将SdBean转换为SdClassContent,对应:SdClassConverter
  5. 在ScrewDriverCode中创建模板渲染需要的上下文SdRenderContent,并由ScrewDriverTemplate进行渲染

数据库操作执行流程

  1. 创建一个实体用于描述代码的类属性,对应:SdBean
  2. 实例化ScrewDriverSQL
  3. 在ScrewDriverSQL中将SdBean装换为SdAbstractTableContent,对应:SdTableContentConverter
  4. 在ScrewDriverSQL中SdDbUtil支出创建表

读取数据库表生成代码执行流程

  1. 由SdDbUtil读取表得到SdTable
  2. 将SdTable转换为SdClassContent,通过SdTableToClassConverter
  3. 添加要生成代码的模板,对应:Map(String, SdTemplate)
  4. 剩余的步骤与生成代码执行流程一致
